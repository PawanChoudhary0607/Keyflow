This folder holds the screenshots and demo GIF referenced in the root
README.md. Add the following files here before publishing:

- `home.png` — home / typing screen in its default state
- `test.png` — a typing test in progress, mid-session
- `settings.png` — the settings panel open (themes, fonts, sound)
- `results.png` — the results screen after a completed test
- `dark-theme.png` — the app in dark mode
- `light-theme.png` — the app in light mode

And in the parent `assets/` folder:

- `demo.gif` — a short screen recording of a typing session

Recommended size: 1440x900 or similar, PNG for screenshots, GIF under
10MB for the demo (trim/compress with a tool like `gifski` if needed).
